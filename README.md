# Reliant Frontend

This is the Next.js 15 frontend for the Reliant Home Health Agency platform.

## Getting Started

1. **Install Dependencies**
   Ensure you are using Node 20+.
   ```bash
   npm install
   ```

2. **Environment Variables**
   The `.env.local` file should have the API URL pointing to your backend:
   ```env
   NEXT_PUBLIC_API_URL=http://localhost:3001
   ```

3. **Start the Development Server**
   ```bash
   npm run dev
   ```

The frontend will be available at `http://localhost:3000`.
